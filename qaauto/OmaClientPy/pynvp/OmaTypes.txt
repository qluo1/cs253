// Text file that specifies enums and their string conversions.
// Corresponding C++ files OmaTypes.[CH] are generated using PERL.
// 
// (The enums here were taken from earlier versions of OmaTypes.H.)
// 
// Syntax mnemonics:
// 	T_		==> tag prefix
// 	V		==> string value unless specified by a
// 			    type qualifier prefix
// 	Fun		==> function tag; can be repeated. The first one
//			    also generates the operator<< for C++ and
//			    toString for Java
// 	eToS		==> enum to string
// 	sToE		==> string to enum
// 	errorStr	==> error string; for invalid enum conversions
// 	errorEnum	==> error enum; for invalid string conversions
// 	<...>		==> required tag/value sequence
// 	[...]		==> optional tag/value sequence
// 	C/C++ comments	==> treated as comments
// 
// Syntax:
// <T_Begin>
// <T_Enum V> [,T_Wrap V]					// (a)
// [T_Fun V_eToS V_sToE V_errorStr_eToS V_errorEnum_sToE]	// (b)
//  .
//  .
//  .
// <enumName_V [= int_V]> [,V] ...				// (c)
//  .
//  .
//  .
// <T_End>
// 
// (a) starts a enum defn; optional #ifdef wrapper for that enum defn
// (b) specifes function names and error return values
// (c) enumeration; followed by a string V corresponding to
//     each T_Fun function tag.
